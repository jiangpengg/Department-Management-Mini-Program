# Department Management Mini Program

部门综合事务管理微信小程序原型，覆盖腾讯会议申请、会议室申请、印章申请、任务推送、科技项目进度管控和审批工作台。

## 当前功能

- 工作台：首页指标、快捷申请、最近申请、项目风险提醒
- 申请：腾讯会议、会议室、印章申请表单
- 审批：申请筛选、通过、驳回模拟操作
- 任务：任务进度、优先级、完成状态
- 项目：科技项目阶段、节点、预算、风险和进度展示
- 我的：权限、流程、台账、订阅消息等管理入口

## 开发方式

1. 使用微信开发者工具导入本仓库目录。
2. AppID 可先使用测试号或游客模式。
3. 当前数据来自 `utils/mock.js`，后续可替换为微信云开发数据库和云函数。
